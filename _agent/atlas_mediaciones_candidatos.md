# Atlas de mediaciones: candidatos para expansión geográfica

**Corpus de partida:** *Roulin: imágenes, mediaciones y circulación visual*  
**Objetivo:** organizar candidatos —viajeros, naturalistas, médicos, dibujantes, ingenieros, militares, editores y miembros de círculos científicos— como posibles entidades y relaciones para una base de datos semántica del proyecto **Atlas de mediaciones**.

Este documento no funciona como una bibliografía general, sino como una primera estructura curatorial para modelar relaciones entre personas, obras, expediciones, regiones, materiales visuales, repositorios digitales y cadenas de mediación histórica.

---

## Criterio de organización

A partir del informe de investigación adjunto, la base más sólida no es una lista general de viajeros, sino un conjunto de **candidaturas obra–autor–expedición–región**. El informe identifica como núcleo documentado cinco casos fuera de Sudamérica dentro de la constelación de colaboradores de la edición de 1836 del *Règne animal*: **Audouin, Milne-Edwards, Deshayes, Blanchard y Valenciennes**. También deja a **d’Orbigny** y **Roulin** como referencias sudamericanas comparativas, y a **Doyère, Dugès, Duvernoy y Laurillard** como nombres de baja prioridad por no tener libros de viaje localizados en esa revisión.

## 8. Notas de modelado para la base semántica

### 8.1. Entidades mínimas recomendadas

Para que estos registros no queden reducidos a bibliografía, conviene modelar al menos las siguientes clases:

- `Persona`
- `Obra`
- `Serie editorial`
- `Expedición`
- `Sección de obra`
- `Lámina / mapa / vista / tipo humano / espécimen visual`
- `Lugar / región`
- `Repositorio digital`
- `Institución científica`
- `Cadena de mediación`
- `Hipótesis curatorial`

### 8.2. Propiedades útiles

- `autor_de`
- `coautor_de`
- `colabora_en`
- `participa_en`
- `comanda`
- `contribuye_a`
- `forma_parte_de`
- `se_publica_en`
- `contiene_láminas_de`
- `representa`
- `describe`
- `se_vincula_con`
- `es_digitalizado_por`
- `tiene_repositorio_digital`
- `funciona_como_caso_comparativo_de`
- `puede_modelarse_como`
- `requiere_verificación`

### 8.3. Estados de evidencia

Se recomienda distinguir explícitamente entre:

- `documentado`: afirmación apoyada por informe, catálogo o repositorio.
- `inferido`: relación razonable a partir de contexto histórico, editorial o institucional.
- `hipótesis`: relación prometedora pero pendiente de verificación.
- `pendiente`: dato aún no localizado o enlace no comprobado.
